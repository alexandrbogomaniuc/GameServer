遊び方
コインを選択してプレイ、勝利を目指そう 各ショットは選択したコインの値でのベットとなります。各ゲームルームは、ロビーで選択した弾丸ごとのコストレンジ（ベット）によってグループに分かれています。同時に発生した賞金は、画面上で合算されてから獲得されます。
各ショットは命中が確定しています スキル不要！発射した攻撃がターゲットに命中しない場合、敵にあたるまで画面の端で跳ね返ります。1度に画面上に発射できる攻撃は発までとなります。ラウンド終了時に命中していない攻撃が発射されたままの場合、その攻撃へのベットは返金されます。命中する前の攻撃が画面上にある場合、ベットを変更することはできません。
各ショットで賞金獲得の可能性があります 全てのショットには、一撃で敵を倒す可能性があります。スペシャルウェポンは標準の砲塔に比べ、より大きな確率で敵を一撃で倒します。敵を倒すとキャッシュペイアウトまたはスペシャルウェポンが手に入ります！

クリティカルヒット
すべてのウェポンは攻撃するごとにランダムにクリティカルヒットとなる可能性があります。ウェポンによるクリティカルヒットは、ウェポンごとに、そして敵ごとに異なります。

スペシャルウェポンはクリティカルヒットの確率が高まり、あらゆる敵にダメージを与えます！クリティカルヒットは、その発射した攻撃から得られる額にマルチプライヤーが適用されます。マルチプライヤーは2倍、3倍、5倍、7倍、10倍となります。砲塔のショットで獲得できるのは2倍マルチプライヤーのみとなります。
インスタントビクトリー
インスタントビクトリーが手に入る確率は動的で、敵のHPと使用する武器により変動します。

インスタントビクトリーは、伝説のボス、地獄の製錬師、欺きのアバター以外のあらゆる敵のHPが十分に低い時、瞬時に倒すことができます。

インスタントビクトリーからはランダムにプライズが手に入ります！
ボーナス賞金！
敵のHPが十分に低く、残りHPより大きなダメージを与えると、ボーナス賞金が手に入ります！

ボーナス賞金は、敵にダメージを与えた際に手に入る標準プライズに加えて獲得できる現金プライズです。

ターゲットをロック
敵をロックオンするには、十字ボタンをクリックした後、敵をクリックしてください。ロックされると選択した敵にターゲット照準が表示されます。プレイヤーが発射したショットは障害物に関係なくロックしたターゲットに当たります。オートロックが有効な場合、敵を倒すと自動的に次に攻撃するターゲットをロックします。

ウェポンを発射
照準が示す位置に向け、その時点で選択しているウェポンを発射します。照準の位置に敵がいない場合、敵に命中するまで弾丸は画面の端で跳ね返ります。このボタンを押し続けると話すまで連続攻撃を続けます。

• ターゲットロックオン：有効な場合、ゲームは選択されているターゲットモードに応じて自動的に画面上で次に攻撃できるターゲットをロックします。
