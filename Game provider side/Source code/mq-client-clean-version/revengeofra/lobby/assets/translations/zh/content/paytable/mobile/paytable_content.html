怎麼玩
選擇遊戲幣玩並贏 每次射擊即為一次投注，使用你所選的遊戲幣值。每間遊戲室均根據在大廳選擇的子彈射程單位花費(投注)金額進行分組。玩家獲得獎勵時，同時贏利將一起添加至螢幕上。
每次射擊都保證擊中 不需要什麼技巧！如果射擊並未擊中目標，射擊將在螢幕上跳飛，直到擊中其他敵人。在特定時間內最多只有個活躍的射擊。如果一回合結束時，射擊並未擊中任何目標，投注金額將退還給玩家。一場戰鬥中仍有活躍射擊時不得變更投注金額。
每次射擊都有可能贏 任何一次射擊都有機會摧毀它所擊中的敵人。與標準砲塔相比，特殊武器有更高的機會摧毀敵人。敵人可以獎勵現金支付或特殊武器！
敵人生命值與勝利！
《終極任務：盜墓迷城》中的每個敵人都有指定的生命值。
敵人的等級範圍從1到3。其生命值隨等級遞增。
使用砲塔或特殊武器攻擊會降低敵人的生命值。 支付金額取決於你對敵人造成的傷害。你使用武器造成的傷害越大，支付金額就越高！ 特殊武器會對所有敵人造成更大傷害，尤其是團隊和傳奇BOSS！

武器 遊戲房間按每個子彈花費分類。從大廳中選擇每個子彈的費用值，以進入該類型的第一個可用房間。

標準基礎武器是砲塔，可以使用熒幕上的十字瞄準線瞄準。砲塔有5個投注等級。
透過增加每次射擊的投注來調高投注等級。砲塔等級越高，射擊支出就越大。

誤用/故障使所有支付和遊戲都無效。
砲塔 基礎武器可以隨時使用。
特殊武器 除基礎武器（砲塔）外，擊中敵人時還有機會掉落特殊武器。這些特殊武器更加強大，可以傷害一個或多個敵人。明智地使用特殊武器：它們在對抗強敵或一群敵人時最有效。特殊武器有更大機會獲得即時贏利和獎金贏利！敵人掉落的特殊武器無需透過投注進行射擊。武器射擊不會從邊緣彈飛，但會瞄準離你點擊位置最近的敵人。未使用的特殊武器在回合結束後轉變為現金獎勵。

在遊戲過程中，可以隨時購買按次付費的特殊武器！特殊武器射擊會根據所選武器增加下注金額。只需在熒幕左側點擊要使用的特殊武器的圖像即可。透過掉落的特殊武器獲得的免費射擊將使用贏得武器時的投注等級，並且在使用免費射擊之前無法更改投注等級。 特殊武器必須要在當前贏利回合中使用。所有未使用的特殊武器都在回合結束後失效。
利用金錢輪盤贏取大贏利！
除傳奇BOSS外，使用任意武器擊中任意敵人都可以在金錢輪盤中獲得即時旋轉。
特殊武器有更高機會觸發金錢輪盤！
暴擊
所有武器發射的每次射擊，都有可能隨機對敵人造成暴擊傷害。不同武器對不同敵人的暴擊是不同的。

特殊武器有更大機會造成暴擊傷害，可以影響任何敵人！暴擊將應用一個翻倍器至從發射的射擊中獲得的獎勵。翻倍器為2倍、3倍、5倍、7倍、10倍。以砲塔射擊僅能獲得2倍翻倍器。
即時贏利
即時贏利的機率是不斷變化的，取決於敵人當前的生命值和所使用的武器。

如果生命值足夠低，即時贏利可以立即擊敗除傳奇BOSS、地獄鍛造師和易爆天神以外的任何敵人。

即時贏利可獲得隨機獎勵！
獎金贏利！
如果敵人的生命值足夠低，而你的射擊造成的傷害極大，則被視為獎金贏利！

除了對敵人造成傷害而獲得的標準獎勵外，獎金贏利還將獲得額外的獎金現金獎勵。
滑鼠符號
十字瞄準線
玩家的光標作為其武器的十字瞄準線。發射的射擊將擊中光標指向的任何地方。
鍵盤指令
空白鍵：向游標所指的任何方向發射目前選用的武器。長按以持續射擊。
右擊
鎖定目標
點擊十字瞄準線，然後點擊鎖定敵人，就會顯示目標十字線。會在所選敵人身上顯示目標十字線。不管有什麼東西擋道，玩家發射的任何射擊都會擊中該目標。如果啟用了自動射擊，遊戲將自動射擊鎖定的目標。如果啟用了自動鎖定，遊戲將自動切換到下一個可用目標。

射擊控制面板
打開射擊控制面板，你可以在此設置自動射擊，自動鎖定和其他選項。

• 鎖定目標：當啟動時，遊戲會根據所選的瞄準模式自動瞄準螢幕上的下一個敵人。

• 自動射擊：當激活時，遊戲會使用活躍武器自動射擊鎖定的目標。如果沒有選擇目標，按住發射按鈕會使得遊戲對著游標指向的任何位置連續射擊。
