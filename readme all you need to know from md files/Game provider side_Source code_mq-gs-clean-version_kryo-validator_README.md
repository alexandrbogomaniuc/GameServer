## Kryo Validator

Утилита для проверки корректности выполнения сериализации и десериализации с использованием Kryo.

<br>

#### Выпуск новой версии.

Для выпуска новой версии необходимо сначала выполнить команду:
`mvn release:prepare`.
Задать версию в соответствии с semver'ом.<br>
После успешного завершения, выполнить:
`mvn release:perform`
